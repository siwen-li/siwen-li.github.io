# 扩展 2

利用递归查询生成连续数字、日期等

利用递归查询生成 1~10 连续数字
=== "connect by 方式"
sql select level from dual connect by level <= 10;

=== "with 方式"
```sql
with t(rn)
as
(select 1 rn from dual
union all
select rn + 1 from t
where rn < 10)

select * from t;
```

利用递归查询生成当前月的连续日期
=== "connect by 方式"
sql select trunc(sysdate,'mm') + level - 1 dt from dual connect by level <= last_day(trunc(sysdate,'mm')) - trunc(sysdate,'mm') + 1;

=== "with 方式"
```sql
with t(rn)
as
(select 1 rn from dual
union all
select rn + 1 from t
where rn < last_day(trunc(sysdate,'mm')) - trunc(sysdate,'mm') + 1)

select trunc(sysdate,'mm') + rn - 1 dt from t;
```
